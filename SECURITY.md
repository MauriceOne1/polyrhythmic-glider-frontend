# Security Policy

## Supported Versions

We currently maintain and support only the latest version of the project.

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue.

You can report it responsibly by contacting:

📧 maurizio [dot] querzoli [at] gmail [dot] com

We will respond within a few days to acknowledge your report.

If the vulnerability is confirmed:

- we will work on a fix as soon as possible
- we will release an update
- we will give you credit (if desired)

Thank you for helping keep Polyrhythmic Glider secure 🙏
